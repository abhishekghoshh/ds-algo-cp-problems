- O(1): Acceptable for tiny inputs.

- O(N): Acceptable for N up to 10^8
- O(NlogN): Acceptable for N up to 10^5
- O(N^2): Acceptable for N up to 10^4
- O(N^3): Acceptable for N up to 10^2
